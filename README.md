## Local Development

1. Setup environment variables (e.g. via Doppler)
2. Run database migration: `pnpm run db:migrate`
3. Start dev server: `pnpm run dev`

## Deployment

Deployment is done locally via `pnpm run ship` (see AGENTS.md).
